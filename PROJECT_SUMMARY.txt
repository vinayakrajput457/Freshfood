## ğŸ”§ Customization Guide

### Change Colors
Edit `:root` in `css/style.css`:
```css
--primary-color: #FF6B35;     /* Change main color */
--secondary-color: #004E89;   /* Change secondary color */
--dark-color: #1a1a1a;        /* Change text color */
```

### Change Products
Edit `products` array in `js/data.js`:
```javascript
const products = [
    {
        id: 1,
        name: "Your Product",
        category: "Your Category",
        price: 9.99,
        // ... more fields
    }
];
```

### Change Promo Codes
Edit `promoCodes` in `js/data.js`:
```javascript
const promoCodes = {
    'YOUR_CODE': { discount: 20, type: 'percentage' }
};
```

### Modify Pricing
Edit these constants in checkout files:
```javascript
DELIVERY_FEE = 3.99;  // Change delivery fee
TAX_RATE = 0.08;      // Change tax rate
```
